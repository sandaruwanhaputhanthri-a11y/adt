“බස් එක නවත්වන්න, නවත්වන්න.” ළමයි කෑගැසුවා.

“මෙතන නවත්වන්න තහනම් පුතේ.

අර බලන්න අර පුවරුව.

ඒ පුවරුවෙන් කියවෙන්නේ වාහන නවත්වන්න එපා කියලයි.”

මැණික් වැව

බස් රථය තරමක් ඉදිරියෙන් නතර කළා.

“දැන් දෙන්නා දෙන්නා පේළියට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

බස් රථය තරමක් ඉදිරියෙන් නතර කළා.

“දැන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

බස් රථය නතර කළයුත්තේ ඉදිරියෙන් නතර කළා.

“දුන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

විලක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ විපුටට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

“දැන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

විලක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ විපුටට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

බස් රථය තරමක් ඉදිරියෙන් නතර කළා.

“දැන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

“දුන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

බස් රථය තරමක් ඉදිරියෙන් නතර කළා.

“දැන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

බස් රථය තරමක් ඉදිරියෙන් නතර කළා.

“දැන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

බස් රථය තරමක් ඉදිරියෙන් නතර කළා.

“දැන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

බස් රථය නතර කළ යුත්තේ ඉදිරියෙන් නතර කළා.

“දැන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

බස් රථය තරමක් ඉදිරියෙන් නතර කළා.

“දැන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

බස් රථය තරමක් ඉදිරියෙන් නතර කළා.

“දැන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

“දුන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.

බස් රථය තරමක් ඉදිරියෙන් නතර කළා.

“දැන් දෙන්නා දෙන්නා පෙළට එන්න.”

ගුරුතුමිය ළමයින් එක්ක කහ ඉර හරහා පාර මාරු වුණා.

ටිකක් දුර ගිහින් ග්‍රහලෝකාගාරයට ඇතුළු වුණා.

ග්‍රහලෝකාගාරයේ චිත්‍රපට ශාලාවක වගේ පුටු තිබුණා.

ළමයි නිශ්ශබ්දව වාඩි වුණා.
